# MyNotes
is an WPF application for viewing, editing, saving, deleting notes in notebooks.

## Installing
You can download the 
[latest release](https://github.com/Sv9ta/MyNotes/releases/download/v1.0.0/MyNotes.exe) and run it.

## Usage
